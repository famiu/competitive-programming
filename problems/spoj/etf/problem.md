# ETF - Euler Totient Function

**Time limit:** 1 s | **Memory limit:** 1536 MB | **Platform**: SPOJ

---

In number theory, the totient $\varphi$ of a positive integer $n$ is defined to be the number of positive integers less than or equal to $n$ that are coprime to $n$.

Given an integer $n$ ($1 \le{} n \le{} 10^6$). Compute the value of the totient $\varphi$.

## Input

First line contains an integer $T$, the number of test cases. ($T \le{} 20000$)

$T$ following lines, each contains an integer $n$.

## Output

$T$ lines, one for the result of each test case.

## Constraints

* $1 \le{} T \le{} 20000$
* $1 \le{} n \le{} 10^6$

## Examples

### Example 1

**Input:**

```
5
1
2
3
4
5
```

**Output:**

```
1
1
2
2
4
```
