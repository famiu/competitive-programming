# COMDIV - Number of common divisors

**Time limit:** 0.6 s | **Memory limit:** 1536 MB | **Platform**: SPOJ

---

You will be given $T$ ($T \le{} 106$) pairs of numbers. All you have to tell is the number of common divisors between the two numbers in each pair.

## Input

First line of input: $T$ (Number of test cases)
In next $T$ lines, each have one pair $A$ $B$ ($0 < A$, $B \le{} 106$)

## Output

One integer describing number of common divisors between two numbers.

## Constraints

* $T \le{} 10^6$
* $0 < A, B \le{} 10^6$

## Examples

### Example 1

**Input:**

```
3
100000 100000
12 24
747794 238336
```

**Output:**

```
36
6
2
```
