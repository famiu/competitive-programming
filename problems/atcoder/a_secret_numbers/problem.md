# A - Secret Numbers

**Time limit:** 2 s | **Memory limit:** 1024 MB | **Platform:** AtCoder

---

You are given a string $S$ consisting of lowercase English letters and digits.

Extract only the digit characters from $S$ and output the string formed by arranging them in their original order.

## Constraints

* $S$ is a string of length between $1$ and $50$, inclusive, consisting of lowercase English letters and digits.

## Input

The input is given from Standard Input in the following format:

```
$S$
```

## Output

Output the answer.

## Examples

### Example 1

**Input:**

```
abc462
```

**Output:**

```
462
```

### Example 2

**Input:**

```
codequeen
```

**Output:**

```

```

### Example 3

**Input:**

```
31415
```

**Output:**

```
31415
```

### Example 4

**Input:**

```
10plus2is12
```

**Output:**

```
10212
```
